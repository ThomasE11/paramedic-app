## 🚀 How To Use These Templates

### Method 1: Download and Fill (Recommended)

1. **Choose** a template that fits your needs
2. **Download** or open the file
3. **Edit** the data:
   - Replace sample student names with your students
   - Update sites to your actual sites
   - Change dates to your semester dates
   - Adjust mentors and shifts
4. **Save** the file
5. **Upload** to the system via:
   - Timetables → Upload Schedule → Upload File

### Method 2: Use as Reference

1. **Open** any template
2. **See** the format and structure
3. **Create** your own file matching the style
4. **Upload** when ready

### Method 3: Quick Start

1. **Use Template #5 (Blank)**
2. **Fill in** your data
3. **Upload** immediately

## 💡 Tips for Best Results

### Student Information
✅ **DO:**
- Include Student IDs for reliable matching
- Use full names as in database
- Keep formatting consistent

❌ **AVOID:**
- Nicknames or abbreviations
- Missing Student IDs
- Special characters in IDs

### Dates
✅ **DO:**
- Use YYYY-MM-DD format (e.g., 2025-01-15)
- Or DD/MM/YYYY (e.g., 15/01/2025)
- Be consistent within the file

❌ **AVOID:**
- Mixing formats (Jan 15 AND 15/01)
- Missing dates
- Invalid dates (e.g., 32/01/2025)

### Site Names
✅ **DO:**
- Use full official names
- Be consistent (same name for same site)
- Include department if relevant

❌ **AVOID:**
- Abbreviations only (unless documented)
- Variations of same site name
- Typos

### Site Types
**Valid types:**
- hospital
- ambulance_station
- community_clinic
- primary_care
- urgent_care
- specialty_clinic

### Shifts
**Common values:**
- day
- night
- rotating
- morning
- evening
- 24-hour

## 📝 Column Descriptions

### Required Columns (Minimum)
1. **Student Name** - Full name of student
2. **Site Name** - Where they're placed
3. **Start Date** - When placement begins
4. **End Date** - When placement ends

### Highly Recommended
5. **Student ID** - For accurate matching
6. **Site Type** - Type of placement site
7. **Week Number** - Week of rotation
8. **Shift** - Day/Night/Rotating

### Optional but Useful
9. **Mentor** - Supervisor/mentor name
10. **Department** - Specific department
11. **Notes** - Additional information

## 🔄 Example Workflow

### Scenario: You have 20 students, 2 rotations

1. **Open** `1-Simple-Table-Template.xlsx`
2. **Delete** sample data (rows 2-11)
3. **Add** your 20 students for Week 3-6
4. **Add** another 20 rows for Week 7-10
5. **Save** as "My_Placements_Spring_2025.xlsx"
6. **Go to** app → Timetables → Upload Schedule
7. **Click** Upload File tab
8. **Select** your file
9. **Click** "Upload & Parse"
10. **Review** the 40 parsed placements
11. **Click** "Create 40 Placements"
12. **Done!** ✅

**Time taken:** ~15 minutes (vs 2+ hours manual entry!)

## 🎨 Customization

### Can I add more columns?
**Yes!** The AI will ignore extra columns, just include the required ones.

### Can I change column order?
**Yes!** AI understands content, not position.

### Can I use different sheet names?
**Yes!** AI reads all sheets.

### Can I include headers in row 1?
**Yes!** Recommended for clarity.

### Can I have multiple week blocks?
**Yes!** Just separate with blank rows or use "Week X-Y" headers.

## 🆘 Troubleshooting

### "Students not matched"
- **Check:** Student ID in template matches database
- **Fix:** Add students to database first OR use exact names

### "Dates look wrong"
- **Check:** Date format is consistent
- **Fix:** Use YYYY-MM-DD format

### "Too many sites created"
- **Check:** Site name spelling is consistent
- **Fix:** Use exact same name for same site

### "Upload failed"
- **Check:** File size (under 10 MB)
- **Check:** File format (xlsx, csv, pdf, jpg, png, txt)
- **Fix:** Save in different format OR reduce file size
