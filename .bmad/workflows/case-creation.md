# Case Creation Workflow

## Purpose
Create realistic, medically accurate paramedic training scenarios

## Steps

### 1. Medical Validation
- Run `/medical-review` to validate clinical accuracy
- Verify treatment protocols match UAE standards
- Check medication dosages
- Confirm difficulty level appropriate

### 2. Vital Sign Design
- Determine initial vitals based on condition
- Set deterioration profile (stable → critical)
- Define reassessment intervals
- Plan treatment responses

### 3. Assessment Design  
- Create 8-12 checklist items
- Assign points (5-25 per item)
- Mark critical items
- Write learning objectives

### 4. Visual Resources
- Select 3-5 relevant images
- Choose 2-3 educational videos
- Link to reference articles
- Verify all URLs work

### 5. Clinical Review
- Peer review by paramedic educator
- Test with student group
- Revise based on feedback
- Final approval

## Output
Complete case ready for case database

## Quality Gates
- Medical accuracy: ✓
- Educational value: ✓
- Technical feasibility: ✓
- Student testing: ✓
